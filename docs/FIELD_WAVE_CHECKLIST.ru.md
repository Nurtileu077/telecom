# OPTIQ — волна полевых и инженерных функций (без 2GIS)

## Отменено
- [x] Импорт 2GIS / OSM как отдельный источник

## Волна 1 — поле
- [x] GPS «Где я» на карте
- [x] Поиск по ID камеры / OLT / муфта / ОРК / кабель
- [x] QR-ссылка на паспорт объекта (`?open=ork:…`)
- [x] Чеклист монтажа ОРК и муфты
- [x] Фото объекта (Supabase Storage `field-photos` или офлайн в JSON)
- [x] Полевой режим `?mode=field` — чеклист и фото без правок сети

## Волна 2 — инженерия
- [x] Тип прокладки кабеля (воздух / канал / грунт) + опоры
- [x] Человекочитаемое имя линии + автоподпись
- [x] Диаграмма стоимости по районам (вкладка «Стоимость»)
- [x] Сценарии A/B — сохранение и таблица сравнения (Инструменты)
- [x] Патч-панель PON на OLT (внутри объекта)
- [x] Read-only ссылка `?mode=view&project=…`
- [ ] Роли в Supabase Auth — отдельно

## Как проверить
1. Открыть ОРК → чеклист + QR → отсканировать на телефоне.
2. Шапка: поле «ID / камера…» → переход на карту.
3. Кабель → имя, прокладка, опоры.
4. Кнопка навигации справа на карте — GPS.
5. Стоимость → столбики по районам при 2+ районах.
